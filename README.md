Weather Dashboard 
One of my first portfolio projects! I built this weather app to practice my frontend skills and get comfortable working with real-world data. You can search any city and instantly see the current weather and a 5-day forecast.
Getting the UI to look clean and readable was the biggest challenge — figuring out how to lay out the cards, make it responsive on mobile, and display the data in a way that actually made sense visually took some trial and error.
What it does

Search any city in the world
Shows current temperature, weather condition, humidity, wind speed, feels-like temp, and visibility
Displays a 5-day forecast with daily highs and lows
Works on both desktop and mobile

How to run it

Get a free API key from openweathermap.org
Open app.js and replace YOUR_API_KEY on line 9 with your key
Open index.html in your browser — that's it, no installs needed!

Built with
